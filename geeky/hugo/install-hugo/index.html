Install it

Hugo comes in a single binary file. Download the archive from releases page to your ~/Downloads directory. The version and checksums in the examples will likely be different, of course.

Calculate the checksum:

openssl sha256 -hex -r ~/Downloads/hugo_0.69.2_macOS-64bit.tar.gz
f58e9312ee292c082c5833a128e751c1d07c5fb0da1b7584c09131f18e2ccf52 */Users/you/Downloads/hugo_0.69.2_macOS-64bit.tar.gz

Verify that the checksum matches the one listed in hugo_0.69.2_checksums.txt file, that can be found on the same releases page. Extract the archive you’ve just verified:

tar zxvf ~/Downloads/hugo_0.69.2_macOS-64bit.tar.gz

Copy the hugo executable to a directory in your $PATH. /usr/local/bin should fit the bill.

sudo cp ~/Downloads/hugo_0.69.2_macOS-64bit/hugo /usr/local/bin/

Make sure the file is globally executable (so far, it has been by default):

sudo chmod 0755 /usr/local/bin/hugo

Verify by running

hugo help

If you see a message like the one below (shortened here), you are done!

hugo is the main command, used to build your Hugo site.

Hugo is a Fast and Flexible Static Site Generator
built with love by spf13 and friends in Go.
